Typography

Typography is all about how text looks.

There are many CSS properties that can be used to customize typography.

text-transform can change the casing of your text.

  • "HeLlO wOrLd" → text-transform: lowercase; → "hello world"
  • "HeLlO wOrLd" → text-transform: uppercase; → "HELLO WORLD"
  • "HeLlO wOrLd" → text-transform: capitalize; → "Hello World"

font-size can change the size of your text.

  • font-size: 10px;
  • font-size: 20px;
  • font-size: 30px;

font-weight can change the boldness of your text.

  • font-weight: 200;
  • font-weight: 400;
  • font-weight: 700;

line-height can change how much space is between your lines.

  • line-height: 10px;
